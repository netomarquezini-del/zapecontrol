# Story 1.1 — Setup do Projeto e Autenticação

**Epic:** 1 — MVP
**Status:** Draft
**Agente:** @dev

## Objetivo
Criar a estrutura base do projeto Next.js com autenticação funcional (login, sessão, middleware de proteção de rotas).

## Acceptance Criteria

- [ ] Projeto Next.js 14 inicializado com App Router, Tailwind CSS, TypeScript
- [ ] shadcn/ui configurado com tema base
- [ ] Supabase client configurado (browser + server)
- [ ] Tabela `users` criada com migration
- [ ] NextAuth.js configurado com CredentialsProvider
- [ ] Página de login funcional (email + senha)
- [ ] Middleware protegendo rotas `/dashboard/*`
- [ ] Sessão JWT com id, email, name, role
- [ ] Layout base do dashboard (sidebar + header) com placeholder
- [ ] Seed com usuário admin padrão (admin@festeja.com / admin123)

## Notas Técnicas
- Usar bcrypt para hash de senha
- JWT session (não database session)
- Middleware em `src/middleware.ts`
- Route groups: `(auth)` para login, `(dashboard)` para área autenticada
