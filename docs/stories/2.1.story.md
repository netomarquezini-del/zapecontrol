# Story 2.1 — Dashboard de Produção

**Epic:** 2 — Core
**Status:** Draft
**Agente:** @dev

## Objetivo
Dashboard visual com métricas de produção, gráficos e indicadores.

## Acceptance Criteria

- [ ] Página `/dashboard` (home) com layout de cards + gráficos
- [ ] Cards resumo no topo:
  - [ ] Produção de hoje (quantidade total + pontos)
  - [ ] Meta de hoje (% atingimento + indicador 🟢🟡🔴)
  - [ ] Produção da semana (total)
  - [ ] Comparativo semana atual vs anterior (↑↓ %)
- [ ] Gráfico de barras: produção diária últimos 7/14/30 dias (toggle)
- [ ] Gráfico de linha: produção vs meta (overlay)
- [ ] Ranking top 10 produtos mais produzidos (período selecionado)
- [ ] Filtros globais: período, produto específico, tipo (embalado/desembalado/todos)
- [ ] Clique em produto → detalhamento com histórico de produção
- [ ] Dados calculados via scoring engine (pontos por regras)
- [ ] Loading skeleton enquanto carrega dados
- [ ] Acesso: todos os perfis

## Notas Técnicas
- Recharts para gráficos
- API routes retornam dados agregados (não raw entries)
- Scoring engine aplicada server-side
- Considerar `useSWR` para refresh automático

## Dependências
- Story 1.4 (lançamentos existentes)
- Story 2.2 (metas — pode desenvolver em paralelo, dashboard funciona sem metas)
