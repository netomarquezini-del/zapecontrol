# Story 3.1 — Exportação de Relatórios

**Epic:** 3 — Complementar
**Status:** Draft
**Agente:** @dev

## Objetivo
Exportar dados de produção e dashboards em Excel e PDF.

## Acceptance Criteria

- [ ] Botão "Exportar" no dashboard e na tela de histórico
- [ ] Opções: Excel (.xlsx) ou PDF
- [ ] Excel — Exportação de produção:
  - [ ] Colunas: data, SKU, produto, tipo, quantidade, pontos
  - [ ] Filtros aplicados respeitados
  - [ ] Sheet formatada com headers em negrito
- [ ] Excel — Exportação de metas:
  - [ ] Colunas: período, meta, realizado, % atingimento
- [ ] PDF — Exportação do dashboard:
  - [ ] Layout do dashboard capturado visualmente
  - [ ] Cabeçalho: FesteJá Control + período + data geração
- [ ] Loading state durante geração
- [ ] Download automático ao finalizar
- [ ] Acesso: Admin e Gerente

## Notas Técnicas
- Excel: `xlsx` (SheetJS) via API route
- PDF: `@react-pdf/renderer` via API route
- Gerar server-side para performance

## Dependências
- Story 2.1 (dashboard)
- Story 2.2 (metas)
