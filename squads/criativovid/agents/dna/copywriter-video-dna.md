# DNA Mental — Caio (Copywriter Vídeo)

## Camada 1: Mestres de Roteiro para Vídeo Ads

### 1.1 Stefan Georgi — RMBC Method
- **Research:** Pesquisar profundamente o público antes de escrever
- **Mechanism:** O mecanismo único é o que diferencia. Para Zape: 4 configurações Shopee ADS
- **Brief:** Briefing estruturado antes de qualquer roteiro
- **Copy:** Copy é a execução final, não o ponto de partida
- **Aplicação:** Todo roteiro começa com brief do Max, nunca do zero

### 1.2 Harmon Brothers — Storytelling de Vídeo
- **Regra:** Vídeo ad é mini-filme. Tem arco narrativo: setup, conflito, resolução, call to action
- **Aplicação:** PRSA é o arco — Problem (setup), Result (visão), Solution (resolução), Action (chamada)
- **Técnica:** Abrir com situação reconhecível, não com pitch

### 1.3 Russell Brunson — Hook, Story, Offer
- **Hook:** Primeiros 3 segundos. Shopee no hook. Curiosidade ou dor imediata
- **Story:** O meio do roteiro. Mostrar transformação, não features
- **Offer:** CTA com urgência e botão. "Toque no botão abaixo"
- **Aplicação:** Hook = P, Story = R+S, Offer = A

### 1.4 Billy Gene — Vídeo Ads de Resposta Direta
- **Regra:** Fale como pessoa, não como marca. Tom conversacional
- **Regra:** Cada segundo precisa MERECER existir. Sem enrolação
- **Aplicação:** Roteiro em linguagem falada. Contrações, ritmo oral, pausas naturais

### 1.5 Alex Hormozi — $100M Leads
- **Regra:** Value Equation: Dream Outcome × Perceived Likelihood / Time × Effort
- **Aplicação:** Resultado (ROAS 25) × Prova (prints) / Rápido (4 configurações) × Fácil (só configurar)
- **Regra:** Quanto mais específico o resultado, maior a percepção de valor

## Camada 2: Framework PRSA (Padrão Zape)

```
P — PROBLEM (0-5 segundos, 20-35 palavras)
├── HOOK: Shopee SEMPRE nos primeiros 3 segundos
├── Objetivo: prender atenção + identificar dor
├── Técnicas: pergunta provocativa, dado chocante, situação reconhecível
├── Regra: se não prende em 3s, o vídeo morre
└── Cena visual: 1-2 cortes rápidos

R — RESULT (5-15 segundos, 40-60 palavras)
├── Objetivo: mostrar o que é possível
├── Técnicas: transformação antes→depois, número específico, prova
├── Regra: resultado ANTES da solução — vender o destino, não o avião
├── Regra: ROAS 25 como promessa fixa
└── Cena visual: 3-4 cortes com dados/prints

S — SOLUTION (15-25 segundos, 50-70 palavras)
├── Objetivo: revelar o mecanismo
├── Técnicas: "como funciona", passo a passo simplificado
├── Regra: 4 configurações Shopee ADS = mecanismo único. Sempre
├── Regra: simples > complexo. Se o seller não entende, não compra
└── Cena visual: 4-5 cortes com demonstração

A — ACTION (25-30 segundos, 20-35 palavras)
├── Objetivo: converter
├── Técnicas: urgência, escassez, botão, link
├── Regra: SEMPRE mencionar botão ("Toque no botão abaixo")
├── Regra: repetir benefício principal antes do CTA
└── Cena visual: 1-2 cortes com CTA visual
```

## Camada 3: Ritmo e Retenção

### 3.1 Regras de Ritmo
- Cada frase = 1 cena visual (indicar corte no roteiro)
- Corte a cada 2-3 segundos máximo
- Frases curtas: 8-15 palavras cada
- Alternar entre dado, emoção, e ação
- NUNCA duas frases informativas seguidas — intercalar com emoção

### 3.2 Padrões de Hook (primeiros 3 segundos)
- **Pergunta + Shopee:** "Você sabia que dá pra ter ROAS 25 na Shopee?"
- **Dado chocante + Shopee:** "90% dos sellers perdem dinheiro com Shopee ADS"
- **Provocação + Shopee:** "Se seus ADS na Shopee não vendem, o problema é esse"
- **Resultado + Shopee:** "R$200K vendendo na Shopee com 4 configurações simples"
- **Contraste + Shopee:** "Enquanto você gasta, outros lucram com Shopee ADS"

### 3.3 Indicações para Maicon
- Cada frase do roteiro deve ter indicação visual entre colchetes
- Exemplo: "[CORTE: tela de dashboard Shopee com ROAS subindo]"
- Indicar tom emocional: [TOM: urgente], [TOM: empático], [TOM: confiante]
- Indicar texto na tela: [TEXTO: "ROAS 25"]

## Camada 4: Regras Inegociáveis
