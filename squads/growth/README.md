# Squad: ZapeAds

## Missão
Dominar a inteligência de anúncios na Shopee — espionagem de concorrentes, criação de criativos de alta conversão e análise de performance de ads.

## Agentes

| Agente | Role | Status |
|--------|------|--------|
| **Hawk** (ad-spy) | Espião Shopee — monitora concorrentes e tendências | ativo |
| **Creative Master** | Criação de criativos e copies para ads | ativo |
| **Ad Spy** | Análise de performance e otimização de campanhas | ativo |

## Estrutura

```
zapeads/
├── assets/          → Assets visuais
├── criativos/       → Criativos gerados
├── dashboard/       → Dashboard v1
├── dashboard-v2/    → Dashboard v2 (Node.js)
├── intel/           → Inteligência competitiva
├── produtos/        → Catálogo de produtos
├── squad/           → Definições de agentes
└── videos/          → Assets de vídeo
```
